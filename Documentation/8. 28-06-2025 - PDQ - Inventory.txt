PDQ - Inventory - Gives lots of information about a computer

- Got familiar with the software.

Some features of PDQ Inventory
- Shows a list of domain computers and allows remote access to a computer.
- shows apps installed on workstations, and shows Windows updates installed on workstations.

- Installed PDQ inventory as a central server on WS16
- Manually added and synced computers from AD
- Managed multiple workstations directly from PDQ inventory.
- Ran commands on workstations from this powerful app.
- Generated several reports for target workstations using the run report tool (application, services, hardware)
- shutdown, restarted computers on the domain from this app

There are lots more on the app.
